# User Input

This folder contains the raw task specification per feature.

Goal: The specification should be clear before the plan. No scattered requirements in chat snippets, no silent assumptions.

## File

```text
user-input/<task-id>.md
```

See `templates/user-input.md` for the template.

## Rules

- Short, concrete, decision-relevant
- Don't prescribe solutions when options are still open
- Reference links, screenshots, or gists here
- If scope changes later, update this document and note it in the plan
